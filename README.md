# RFCs — Real Life Org

Transparente Entscheidungsfindung nach dem Konsent-Prinzip.

## Wie funktioniert's?

1. Kopiere das [Template](rfcs/template.md)
2. Erstelle einen Branch, schreib deinen Vorschlag
3. Öffne einen PR
4. Das Team hat 7 Tage für Review nach dem Konsent-Prinzip
