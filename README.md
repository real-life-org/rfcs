1. Lies [RFC-0001](rfcs/0001-rfc-prozess.md) — dort ist der Prozess beschrieben
2. Kopiere das [Template](rfcs/template.md)
3. Schreib deinen Vorschlag, öffne einen PR
4. Das Team hat 7 Tage für Review
